Seção 2 - Rankings de Tecnologias

Tecnologia Descrição
HTML É a base para organizar conteúdos como textos, imagens, links, tabelas e formulários, permitindo que os navegadores interpretem e exibam as informações na internet.
CSS É responsável por controlar cores, fontes, tamanhos, espaçamentos, posições e o layout dos elementos criados com HTML, tornando as páginas mais organizadas e visualmente atraentes.
JavaScript É responsável por controlar cores, fontes, tamanhos, espaçamentos, posições e o layout dos elementos criados com HTML, tornando as páginas mais organizadas e visualmente atraentes.
Python É utilizada para desenvolver aplicações, sistemas, inteligência artificial, análise de dados, automações e também pode ser usada no backend para criar servidores e aplicações web.

Seção 3 - Glossário Técnico

HTML
Linguagem de marcação usada para estruturar páginas web.
CSS
Linguagem usada para definir aparência e estilos.
JavaScript
Linguagem usada para criar interatividade no frontend e backend.
HTTP
Protocolo de comunicação usado para transferir dados entre navegadores e servidores na internet.
DOM
Modelo que representa a estrutura de uma página HTML, permitindo que o JavaScript acesse e altere elementos da página.
DNS
Serviço que traduz nomes de sites em endereços IP, permitindo o acesso aos recursos na internet.
